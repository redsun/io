CSS盒子/浮动/定位

作者 JIN
发表于 2022-02-06 更新于 2022-04-28 3 分钟阅读

盒子

  1. 盒子由width,height,padding,border组成
  2. 宽度 = width + 左右padding + 左右border
  3. 高度 = height + 上下padding + 上下border

padding

  1. 内容到border之间的空隙
  2. padding: 10px 20px 30px 40px , 上右下左，顺时针
  3. padding: 10px 20px 30px, 上 左右 下， 顺时针，缺哪个和对位一样(10px 20px 30px x)
  4. padding: 10px 20px, 上下 左右 顺时针，缺哪个和对位一样(10px 20px x y)
1
2
3
4
5

padding: 20px 0;
padding: 30px 20px 10px;

padding: 20px
padding-bottom: 0

margin

  1. 盒子之间的距离
  2. 竖直方向有塌陷，取大值
  3. 一些元素默认有margin，需要清除
  4. margin: 10px 20px 30px 40px 和padding一样
  5. 盒子水平居中，margin: 0 auto;
  6. box-sizing: border-box; 盒子内缩，width和height就是实际宽高
1
2
3

/* 删除默认margin padding */
* {padding:0; margin:0;}

行内元素和块级元素转换

块不设置width自动撑满一行如(div,section,p,h,li,ul,ol),行内width自动收缩如(a,span,b,i u),行内块可设置width，还能并排如(img,input)

  1. display:block; 转块
  2. display:inline; 转行内
  3. display: inline-block; 转行内块

显示隐藏

  1. display: none; 节点位置不保留
  2. visibility:hidden; 节点位置保留

浮动

  1. 要浮动，并排的盒子都需要浮动 float:left | right
  2. 父盒子宽度要够，不然掉下来了
  3. 浮动的元素可以设置宽高，已经脱离文档流

清除浮动

  1. 父盒子设置overflow:hidden
  2. 给后面的父盒子设置clear:both
  3. 使用::after给盒子添加最后一个子元素，给该元素设置clear:both
  4. 两个盒子之间盖一堵墙，设置clear:both

相对定位

渲染的是影子，原始位置不变。left,right,top,bottom, 微调元素位置，同时可以做绝对定位的参考盒子。

1
2
3

position: relative;
top:10px
left:10px

绝对定位

通过坐标进行精准定位。脱离文档流，释放自己原有位置。优先找离自己最近有相对定位属性的父盒子做参考，找不到就是浏览器左上角。 常用于做压盖，遮罩等。

固定定位

总是固定在某个位置，不管页面是否滚动，脱离文档流，

1
2

right:100px;
bottom:100px;
