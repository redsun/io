Vapor安装

作者 JIN
发表于 2022-03-19 更新于 2022-04-12 1 分钟阅读

Vapor是Swift语言开发的Web框架。拿来学学，既能学习Swift又能学学后台开发，不错的选择~

安装

1

brew install vapor

新建项目

1
2

vapor new todo #创建todo项目，是否使用fluent，选择数据库，是否需要leaf...
vapor new todo -n #默认生成项目

运行

1

swift run or vapor run  #会下载SPM依赖包

访问

1

http://127.0.0.1:8080

由于网络因素，下载SPM依赖大概率会失败，Xcode不能设置代理，那么咱还可以这样做

1
2
3
4

#前提终端已配置好代理，通过下面命令去下载
xcodebuild -resolvePackageDependencies -scmProvider system 
# -y 打开项目
vapor xcode -y

遇到问题

1
2

#清理一下
vapor clean -u -s -g
开发, 后台
vapor swift
