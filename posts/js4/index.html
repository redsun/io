元字符

元字符功能
\d匹配一个数字
\D匹配一个非数字字符
\w匹配一个单字字符(字幕，数字或下划线)
\W匹配一个非单字字符
\s匹配一个空白字符，包括空格，制表符和换行符
.任意字符
^匹配开头
$匹配结尾
1
2
3

var reg = /^\d{3}-\d{4}-\d{3}$/
var str = '123-4567-890'
console.log(reg.test(str))  // true

字符转义

1
2
3
4

/^.$/   // 检查字符是不是任意字符
/^\.$/  // 检查字符串是不是.
/^\\$/  // 检查字符串是不是\
// 不管一个符号有没有特殊意义，都可以加\保证表达的是符号本身

[] - ^

匹配[]中任意一个, - 指定范围, ^取反

\d[0-9]
\d[^0-9]
\w[a-zA-Z0-9_]
\W[^a-zA-Z0-9_]
1
2
3

var reg = /^[a-zA-Z]\d{3}/
var str = "a123"
console.log(reg.test(str))

量词

*匹配前一个表达式0次或者多次{0,}
+匹配前一个表达式1次{1,}
?匹配前一个表达式0次或者1次{0,1}
{n}n是一个正整数，匹配了前面一个字符刚好出现n次 
{n,}n是一个正整数，匹配前一个字符至少出现了n次 
{n,m}n和m都是整数，匹配前一个字符至少出现n次，最多m次
